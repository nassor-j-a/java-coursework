# Exercise 4 – Count Change

## Problem

Write a Java program that calculates the total amount of money represented by quarters, dimes, nickels, and pennies.

---

## Algorithm

1. Read the number of quarters.
2. Read the number of dimes.
3. Read the number of nickels.
4. Read the number of pennies.
5. Convert each denomination into pennies.
6. Calculate the total.
7. Display the value in dollars.

---

## Formula

totalPennies =
quarters × 25 +
dimes × 10 +
nickels × 5 +
pennies

dollars = totalPennies / 100

remainingPennies = totalPennies % 100

---

## Example

Input

Quarters : 3

Dimes : 2

Nickels : 1

Pennies : 6

Output

$1.06
