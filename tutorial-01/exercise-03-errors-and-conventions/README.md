# Exercise 3 – Errors and Conventions

## Objective

Classify errors found in Java programs.

---

## Error Categories

### Syntax Errors

Errors detected by the compiler.

Example

Missing semicolon

Incorrect class name

Misspelled keywords

---

### Runtime Errors

Errors that occur while the program is running.

Example

Division by zero

NullPointerException

---

### Logic Errors

The program runs successfully but produces an incorrect result.

Example

Using the wrong mathematical formula.
