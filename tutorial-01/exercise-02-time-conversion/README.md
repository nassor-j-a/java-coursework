# Exercise 2 – Time Conversion

## Problem

Write an algorithm that converts a time value in seconds into hours, minutes, and remaining seconds.

---

## Algorithm

1. Read total seconds.
2. Calculate hours.
3. Calculate remaining seconds.
4. Calculate minutes.
5. Calculate remaining seconds.
6. Display the result.

---

## Formula

hours = seconds / 3600

remaining = seconds % 3600

minutes = remaining / 60

seconds = remaining % 60

---

## Sample Input

3671

## Sample Output

Hours : 1

Minutes : 1

Seconds : 11

---

- Modulus operator
